# Generating Venn Diagrams

## Issue

## Installation
Ensure the following dependencies are installed before running the script:

- **R** (Required for generating diagrams)
- **ImageMagick** (Used to crop excess whitespace)

## Running the Script
The script uses `venn_diagram.r` to generate the Venn diagrams.  

To generate the diagrams, run:  
```sh
./generate_images.sh
